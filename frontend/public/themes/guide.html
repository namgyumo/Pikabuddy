1. 테마 JSON 구조 (5계층)

PikaBuddy 테마는 .pikabuddy-theme.json 파일 하나에 5개 계층이 모두 담깁니다.

{
  "name": "My Theme",         // 테마 이름 (필수)
  "version": 1,                  // 항상 1 (필수)

  // [계층 1] GUI 변수 — 47개 CSS 변수
  "variables": {
    "--primary": "#A855F7",
    "--surface": "#0B0E1A",
    ...
  },

  // [계층 2] Custom CSS — 세밀한 스타일 오버라이드
  "customCSS": ".card:hover { box-shadow: 0 0 20px rgba(168,85,247,0.3); }",

  // [계층 3] 애니메이션 프리셋
  "animation": "particles",

  // [계층 4] 상시 이펙트 — layer로 겹침 순서 제어
  "effects": {
    "starfield": { "enabled": true, "params": { "speed": 0.75 }, "layer": 0 },
    "aurora":    { "enabled": true, "params": { ... }, "layer": 1 },
    "glow":      { "enabled": true, "params": { ... }, "layer": 2 }
  },

  // [계층 5] 이벤트 트리거 — 정답/오답/레벨업 시 발동
  "triggers": {
    "onCorrect": ["confetti", "correctBounce"],
    "onWrong":    ["wrongShake"],
    "onLevelUp": ["levelUpCelebration"]
  }
}
필수 필드: name, version, variables (최소 1개 이상)
선택 필드: customCSS, animation, effects, triggers

3. Custom CSS 작성법

customCSS 필드에 CSS를 넣으면 페이지에 직접 주입됩니다.

보안 제한: @import, expression(), javascript:, <script> 태그는 자동 제거됩니다.

자주 쓰는 셀렉터

셀렉터대상
.card, .assignment-card, .note-card카드 컴포넌트
.card:hover카드 호버
h1, h2, h3, .page-title제목
.btn-primary, button[class*="primary"]주 버튼
.sidebar, [class*="sidebar"]사이드바
::-webkit-scrollbar-thumb스크롤바
::selection텍스트 선택 영역

인기 CSS 패턴

그라디언트 텍스트

h1, h2 {
  background: linear-gradient(135deg, #EC4899, #A855F7);
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  background-clip: text;
}

유리 효과 카드

.card {
  background: rgba(255, 255, 255, 0.1) !important;
  backdrop-filter: blur(16px) saturate(1.4);
  border: 1px solid rgba(255, 255, 255, 0.1) !important;
}

네온 글로우 버튼

.btn-primary {
  box-shadow: 0 0 15px rgba(168, 85, 247, 0.4),
              inset 0 1px 0 rgba(255, 255, 255, 0.1);
}

그라디언트 스크롤바

::-webkit-scrollbar-thumb {
  background: linear-gradient(180deg, #A855F7, #6366F1) !important;
  border-radius: 10px;
}

4. 애니메이션 프리셋

animation 필드에 다음 ID 중 하나를 지정합니다.

ID이름설명
"none"없음애니메이션 비활성화
"particles"떠다니는 입자부유하는 파티클
"rain"비빗줄기 효과
"sparkle"반짝임별 반짝임
"snow"눈눈 내리는 효과
"gradient"그라디언트 흐름색상이 흐르는 배경

6. 레이어 시스템 (z-index)

이펙트가 겹칠 때 layer 값으로 렌더링 순서를 제어합니다. 낮을수록 뒤(배경), 높을수록 앞(전경).

커서 효과 (cursorGlow, mouseTrail)layer: 3
요소 스타일 (glow, glassMorphism)layer: 2
중간 효과 (aurora, fadeInScroll)layer: 1
배경 효과 (starfield, particles)layer: 0
"effects": {
  "starfield": { "enabled": true, "params": { ... }, "layer": 0 },  // 가장 뒤
  "aurora":    { "enabled": true, "params": { ... }, "layer": 1 },  // 중간
  "glow":      { "enabled": true, "params": { ... }, "layer": 2 },  // 앞
  "cursorGlow":{ "enabled": true, "params": { ... }, "layer": 3 }   // 가장 앞
}

7. 트리거 시스템 (이벤트 이펙트)

게이미피케이션 이펙트(confetti, wrongShake 등)는 항상 켜져있는 게 아니라 특정 이벤트에 반응합니다.

사용 가능한 이벤트

이벤트발동 시점추천 이펙트
onCorrect정답 제출 시confetti, correctBounce, xpGain, rippleClick
onWrong오답 제출 시wrongShake, glitchText
onLevelUp레벨 상승 시levelUpCelebration, confetti
onBadge뱃지 획득 시badgeUnlock
onStreak연속 정답 시streakFire, comboCounter
onCombo콤보 달성 시comboCounter
onRankUp티어 승급 시rankUpAnimation
onDaily일일 보상 시dailyReward
"triggers": {
  "onCorrect": ["confetti", "correctBounce"],
  "onWrong":   ["wrongShake"],
  "onLevelUp":["levelUpCelebration", "confetti"],
  "onBadge":  ["badgeUnlock"]
}

8. 성능 자동 조절

PikaBuddy 테마 엔진은 자동 성능 관리 기능이 내장되어 있습니다.

통합 애니메이션 루프

Canvas 이펙트(particles, starfield, aurora, matrixRain, bubbles)는 하나의 requestAnimationFrame 루프에서 함께 처리됩니다. 이펙트 5개를 켜도 RAF는 1개만 돌아갑니다.

FPS 자동 감소

실시간 FPS를 측정하여 30fps 이하로 떨어지면 파티클 밀도를 자동 축소합니다 (최소 25%). 50fps 이상으로 회복되면 점진적으로 복원합니다.

탭 비활성화 일시정지

브라우저 탭이 비활성화되면 모든 이펙트 렌더링이 자동 일시정지되고, 다시 활성화되면 재개됩니다.

CSS 이펙트 우선

glow, glassMorphism, gradientBorder, cherryBlossom, fogMist 등은 CSS 애니메이션으로 처리되어 GPU 가속을 활용합니다.

이펙트를 많이 켤수록 파티클이 자동으로 줄어듭니다. 저사양 기기에서도 걱정 없이 사용하세요.

9. 테마 만들기 / 가져오기 / 공유하기

방법 1: GUI 에디터 (권장)

  1. 설정 > 테마에서 "만들기" 클릭
  2. 4개 탭(GUI / JSON / CSS / Effects)에서 편집
  3. 저장하면 자동으로 커스텀 테마로 등록

방법 2: JSON 파일 가져오기

  1. .pikabuddy-theme.json 파일 준비
  2. 설정 > 테마 > 하단 업로드 영역에 드래그 또는 클릭
  3. 자동 검증 후 테마가 추가됩니다

방법 3: 프리셋에서 시작

  1. 설정 > 테마 > "프리셋 커스텀 테마" 섹션에서 원하는 프리셋 클릭
  2. 적용된 후 편집 아이콘으로 수정

테마 공유하기

  1. 테마 에디터 > JSON 탭에서 JSON 복사
  2. .pikabuddy-theme.json 파일로 저장
  3. 친구에게 파일 공유 → 가져오기로 적용

11. 디자인 팁

색상 대비 확인

--on-surface(텍스트)와 --surface(배경) 사이에 충분한 대비(4.5:1 이상)를 유지하세요. 다크 테마에서 특히 중요합니다.

이펙트는 3~5개가 적절

상시 이펙트는 3~5개, 트리거는 원하는 만큼 넣어도 됩니다. 상시 이펙트가 너무 많으면 산만해집니다.

라이트/다크 테마 구분

다크 테마: --surface를 어두운 색, --on-surface를 밝은 색으로. shadow는 primary 색 기반 글로우로.
라이트 테마: --surface를 밝은 색, --on-surface를 어두운 색으로. shadow는 검정 기반 일반 그림자로.

Custom CSS는 보완용

variables로 대부분 커버 가능합니다. Custom CSS는 카드 호버, 그라디언트 텍스트, 스크롤바 등 세밀한 조정에만 사용하세요.

layer 권장 배치

0: 배경 (starfield, particles, aurora)
1: 중간 (fadeInScroll, parallaxScroll, drawBorder)
2: 요소 (glow, glassMorphism, softShadow)
3: 커서/텍스트 (cursorGlow, neonText, mouseTrail)

포인터 학습 노트

C 언어의 핵심 개념 및 메모리 관리 심화

1. 포인터의 기초

포인터는 특정한 데이터가 저장된 메모리의 주소값을 가지는 변수입니다. 일반적인 변수가 값을 저장한다면, 포인터 변수는 그 값이 어디에 있는지를 가리킵니다.

// 변수 선언
int num = 10;
// 포인터 변수 선언
int *ptr = #

위의 코드에서 int ptr = # 라고 작성하면 에러가 발생합니다. 포인터 타입을 명시하기 위해서는 자료형 뒤에 별표(*)를 붙여야 합니다.

2. 메모리 직접 접근

포인터를 사용하면 메모리에 직접 접근하여 값을 수정할 수 있습니다. 이것은 하드웨어 제어에 필수적이지만, 동시에 매우 위험한 작업일 수 있습니다. 잘못된 주소를 참조하게 되면 세그멘테이션 오류(Segmentation Fault)가 발생합니다.

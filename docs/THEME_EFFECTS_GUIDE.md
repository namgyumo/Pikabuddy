# PikaBuddy 커스텀 테마 이펙트 가이드북

## 1. 배경 효과

### `addParticles(color, count)`
화면에 부유하는 파티클을 생성합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `color` | `string` | `var(--primary)` | 파티클 색상 (CSS 색상값) |
| `count` | `number` | `30` | 파티클 개수 (10~100) |

**미리보기**: 반투명한 원형 입자들이 화면을 천천히 떠다님

---

### `addStarfield(speed, density)`
별이 다가오듯 흐르는 우주 배경을 생성합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `speed` | `number` | `1` | 이동 속도 (0.5~3) |
| `density` | `number` | `100` | 별 밀도 (50~300) |

**미리보기**: 화면 중심에서 바깥으로 별들이 흐르는 워프 효과

---

### `addAurora(colors)`
오로라/북극광 물결 효과를 배경에 추가합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `colors` | `string[]` | `["#00ff87", "#60efff", "#ff00e5"]` | 오로라 색상 배열 (2~5개) |

**미리보기**: 화면 상단에서 부드럽게 물결치는 반투명 오로라 띠

---

### `addMatrixRain(color)`
코드 문자가 떨어지는 매트릭스 스타일 배경을 생성합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `color` | `string` | `"#00ff00"` | 문자 색상 |

**미리보기**: 세로 열로 코딩 문자(0, 1, 한글, 카타카나 등)가 떨어짐

---

### `addBubbles(color, count)`
투명한 거품이 아래에서 위로 떠오르는 효과입니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `color` | `string` | `var(--primary)` | 거품 색상 |
| `count` | `number` | `20` | 거품 개수 (5~50) |

**미리보기**: 다양한 크기의 반투명 원이 천천히 떠오르며 상단에서 사라짐

---

### `addCherryBlossom()`
벚꽃잎이 하늘에서 떨어지는 효과입니다.

| 파라미터 | 없음 | | |
|---------|------|--------|------|

**미리보기**: 분홍색 꽃잎 모양이 좌우로 흔들리며 떨어짐

---

### `addAutumnLeaves()`
낙엽이 바람에 흩날리며 떨어지는 효과입니다.

| 파라미터 | 없음 | | |
|---------|------|--------|------|

**미리보기**: 주황/갈색/빨간 나뭇잎이 회전하며 떨어짐

---

### `addLightning(frequency)`
화면에 간헐적으로 번개가 번쩍이는 효과입니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `frequency` | `number` | `5` | 번개 빈도 (초 단위 간격, 3~15) |

**미리보기**: 화면 전체가 잠깐 밝아졌다가 돌아오는 번개 섬광

---

### `addFogMist(density)`
화면 하단 또는 전체에 안개/미스트 효과를 추가합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `density` | `number` | `0.5` | 안개 밀도 (0.1~1.0) |

**미리보기**: 반투명 흰색/회색 안개가 천천히 흐름

---

## 2. 배경 패턴

### `addDotGrid(color, spacing)`
규칙적인 도트 그리드 패턴을 배경에 추가합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `color` | `string` | `var(--border)` | 도트 색상 |
| `spacing` | `number` | `24` | 도트 간격 (px, 12~48) |

**미리보기**: 균일하게 배치된 작은 점들의 격자

---

### `addNoiseTexture(opacity)`
미세한 노이즈/그레인 질감을 오버레이합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `opacity` | `number` | `0.05` | 노이즈 불투명도 (0.01~0.2) |

**미리보기**: 사진 필름 같은 미세한 입자 질감이 전체 화면에 덧씌워짐

---

### `addGeometricPattern(shape)`
기하학 도형이 반복되는 패턴 배경을 생성합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `shape` | `"triangle" \| "hexagon" \| "diamond" \| "circle"` | `"hexagon"` | 도형 모양 |

**미리보기**: 선택한 도형이 반복 배치된 반투명 기하학 패턴

---

### `addMouseGradient(colors)`
마우스 커서 위치를 따라다니는 그라데이션 배경입니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `colors` | `string[]` | `[var(--primary), var(--tertiary)]` | 그라데이션 색상 |

**미리보기**: 마우스 위치 중심으로 방사형 그라데이션이 부드럽게 따라옴

---

## 3. 요소 스타일

### `addGlow(color, intensity)`
주요 UI 요소(카드, 버튼)에 발광 효과를 추가합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `color` | `string` | `var(--primary)` | 발광 색상 |
| `intensity` | `number` | `0.5` | 발광 강도 (0.1~1.0) |

**적용 대상**: `.card`, `.btn-primary`, 주요 컨테이너
**미리보기**: 요소 주변에 은은한 빛번짐 (box-shadow)

---

### `addGlassMorphism(blur, opacity)`
유리/프로스트 효과를 카드 및 패널에 적용합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `blur` | `number` | `12` | 블러 강도 (px, 4~24) |
| `opacity` | `number` | `0.15` | 배경 불투명도 (0.05~0.5) |

**적용 대상**: `.card`, `.modal`, `.sidebar`
**미리보기**: 뒤의 콘텐츠가 흐릿하게 비치는 반투명 유리 패널

---

### `addGradientBorder(colors)`
요소 테두리에 그라데이션 효과를 적용합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `colors` | `string[]` | `[var(--primary), var(--tertiary)]` | 그라데이션 색상 배열 |

**적용 대상**: `.card`, `input:focus`
**미리보기**: 테두리가 여러 색상으로 흐르는 그라데이션

---

### `addCardTilt(intensity)`
카드에 마우스를 올리면 3D로 기울어지는 효과입니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `intensity` | `number` | `10` | 최대 기울기 각도 (5~20) |

**적용 대상**: `.card`
**미리보기**: 마우스 위치에 따라 카드가 입체적으로 기울어짐 (perspective + rotateX/Y)

---

### `addSoftShadow(depth)`
요소에 부드러운 다층 그림자를 적용합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `depth` | `number` | `3` | 그림자 깊이 단계 (1~5) |

**적용 대상**: `.card`, `.modal`
**미리보기**: 여러 겹의 은은한 그림자가 자연스러운 깊이감 생성

---

### `addDrawBorder()`
요소에 호버 시 테두리가 그려지는 애니메이션입니다.

| 파라미터 | 없음 | | |
|---------|------|--------|------|

**적용 대상**: `.card`, `button`
**미리보기**: 호버 시 왼쪽 상단부터 시계 방향으로 테두리 라인이 그려짐

---

## 4. 텍스트 효과

### `addGlitchText()`
제목 텍스트에 디지털 글리치/깨짐 효과를 추가합니다.

| 파라미터 | 없음 | | |
|---------|------|--------|------|

**적용 대상**: `h1`, `h2`, 페이지 타이틀
**미리보기**: 텍스트가 간헐적으로 RGB 분리되며 떨리는 효과

---

### `addRainbowText()`
텍스트에 무지개 그라데이션 색상을 적용합니다.

| 파라미터 | 없음 | | |
|---------|------|--------|------|

**적용 대상**: 주요 타이틀, 강조 텍스트
**미리보기**: 텍스트 색상이 무지개 그라데이션으로 흐르며 애니메이션

---

### `addTextScramble()`
텍스트가 등장할 때 랜덤 문자에서 해독되듯 나타나는 효과입니다.

| 파라미터 | 없음 | | |
|---------|------|--------|------|

**적용 대상**: 페이지 타이틀, 중요 알림 텍스트
**미리보기**: "■◆▲" → "피카버디" 처럼 문자가 하나씩 맞춰지는 디코딩 애니메이션

---

### `addWavyText()`
텍스트 글자가 물결치듯 위아래로 움직이는 효과입니다.

| 파라미터 | 없음 | | |
|---------|------|--------|------|

**적용 대상**: 로고, 섹션 타이틀
**미리보기**: 각 글자가 시차를 두고 부드럽게 위아래로 흔들림

---

### `addNeonText(color)`
텍스트에 네온사인 발광 효과를 추가합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `color` | `string` | `var(--primary)` | 네온 발광 색상 |

**적용 대상**: 타이틀, 강조 텍스트
**미리보기**: 텍스트 주변에 네온 빛이 퍼지고 은은하게 깜빡임

---

## 5. 인터랙션

| 파라미터 | 타입 | 기본값 | 설명 |

**미리보기**: 클릭한 지점에서 원형 물결이 확장되며 사라짐 (Material Design ripple)

## 6. 커서 효과

### `addMouseTrail(shape, color)`
마우스 이동 경로에 궤적을 남기는 효과입니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `shape` | `"dot" \| "star" \| "spark"` | `"dot"` | 궤적 모양 |
| `color` | `string` | `var(--primary)` | 궤적 색상 |

**미리보기**: 마우스를 움직이면 뒤에 작은 도형들이 따라오다가 페이드아웃

---

### `addCursorGlow(color, size)`
커서 주변에 빛나는 원형 후광을 추가합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `color` | `string` | `var(--primary)` | 후광 색상 |
| `size` | `number` | `200` | 후광 크기 (px, 100~400) |

**미리보기**: 커서 위치에 부드러운 방사형 빛이 배경 위에 표시

---

### `addCustomCursor(shape)`
기본 커서를 커스텀 모양으로 변경합니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `shape` | `"crosshair" \| "star" \| "heart" \| "pikachu"` | `"crosshair"` | 커서 모양 |

**미리보기**: 기본 화살표 대신 선택한 모양의 커서 표시

---

### `addClickExplosion(emoji)`
클릭 시 해당 위치에서 이모지가 폭발하듯 퍼지는 효과입니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `emoji` | `string` | `"⚡"` | 폭발 이모지 |

**미리보기**: 클릭 지점에서 선택한 이모지가 8방향으로 퍼졌다가 사라짐

---

### `addTrailEmoji(emoji)`
마우스 이동 시 이모지가 꼬리처럼 따라다니는 효과입니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `emoji` | `string` | `"✨"` | 꼬리 이모지 |

**미리보기**: 마우스 경로를 따라 이모지가 하나씩 생겼다가 페이드아웃

---

## 7. 애니메이션

### `addTypewriterTitle()`
페이지 타이틀이 타이핑되듯 한 글자씩 나타나는 효과입니다.

| 파라미터 | 없음 | | |
|---------|------|--------|------|

**적용 대상**: 페이지 최상단 `h1` 타이틀
**미리보기**: 커서가 깜빡이며 글자가 하나씩 입력되는 타이핑 애니메이션

---

### `addFadeInScroll()`
스크롤 시 화면에 들어오는 요소가 페이드인되는 효과입니다.

| 파라미터 | 없음 | | |
|---------|------|--------|------|

**적용 대상**: `.card`, 섹션 컨테이너
**미리보기**: 스크롤하면 아래에서 위로 올라오며 투명→불투명 전환

---

### `addParallaxScroll(intensity)`
스크롤 시 배경과 전경이 다른 속도로 움직이는 패럴랙스 효과입니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `intensity` | `number` | `0.3` | 패럴랙스 강도 (0.1~0.8) |

**미리보기**: 배경이 전경보다 느리게 스크롤되어 깊이감 생성

---

### `addPulseElement(selector)`
지정 요소가 맥박처럼 주기적으로 확대/축소되는 효과입니다.

| 파라미터 | 타입 | 기본값 | 설명 |
|---------|------|--------|------|
| `selector` | `string` | `".notification-badge"` | 대상 CSS 선택자 |

**미리보기**: 요소가 1→1.05→1 크기로 부드럽게 반복 확대/축소

---

### `addCountUp()`
숫자가 0에서 목표값까지 카운트업되는 애니메이션입니다.

## 구현 노트

### 보안 원칙
- 모든 이펙트는 **사전 정의된 함수**로만 실행 가능
- 사용자 입력은 **파라미터 값**으로만 제한 (타입 검증 필수)
- CSS injection, `eval()`, 인라인 스크립트 등 **일체 금지**
- 색상 파라미터는 hex/rgb/CSS 변수만 허용, 나머지 문자열 필터링

### UI 설계 가이드
- 각 이펙트는 **토글 스위치**로 활성화/비활성화
- 파라미터가 있는 이펙트는 토글 아래에 **슬라이더/컬러피커/드롭다운** 표시
- **실시간 미리보기**: 토글/파라미터 변경 즉시 반영
- 이펙트끼리 **충돌 방지** 로직 필요 (예: 배경 효과 동시 3개 이상 시 성능 경고)
- 카테고리별 **아코디언 그룹**으로 정리

### 성능 고려사항
- 배경 효과는 `requestAnimationFrame` 기반으로 최적화
- 파티클 수, 애니메이션 복잡도에 **상한선** 설정
- 모바일 기기 감지 시 자동으로 파티클 수 50% 감소
- `IntersectionObserver` 활용하여 화면 밖 애니메이션 일시정지
- 이펙트 활성화 시 **FPS 모니터링**, 30fps 이하 시 자동 경고

### 저장 구조 (localStorage)
```json
{
  "themeEffects": {
    "addParticles": { "enabled": true, "params": { "color": "#6C5CE7", "count": 30 } },
    "addGlow": { "enabled": true, "params": { "color": "#6C5CE7", "intensity": 0.5 } },
    "addConfetti": { "enabled": true, "params": {} },
    ...
  }
}
```

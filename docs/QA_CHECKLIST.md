1. 테스트 계정 3개 준비: 교수, 학생, 개인
2. 강의 1개 이상 생성 + 학생 초대 완료
3. 각 타입 과제 1개씩 생성 (코딩, 글쓰기, 퀴즈)
4. 노트 5개 이상 작성 (AI 분석 완료 상태)
5. 팀 1개 이상 생성 + 팀원 배정

### 진행 순서 (권장)
1. **섹션 20** (신규 수정 집중 QA) — 최우선
2. **섹션 1~2** (인증/프로필) — 기본 동작 확인
3. **섹션 3~4** (강의/과제) — 핵심 기능
4. **섹션 7~9** (노트/코멘트/그래프) — 주요 신기능
5. **섹션 5~6** (시험/팀) — 복잡 시나리오
6. **섹션 10~12** (메신저/알림/AI) — 실시간 기능
7. **섹션 13~19** (대시보드/자료/기타) — 나머지
